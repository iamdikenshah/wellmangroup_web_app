# Project: Wellman Group - Static Single Page Website

## Objective
Create a modern, professional **static single-page company website** for **Wellman Group**.

The site must represent the **latest company information from the company profile PDF**.  
The existing website is outdated and should only be used for **visual inspiration**, not content.

Primary source of truth: **Company Profile PDF**

Target audience:
- Hospitals
- Healthcare infrastructure planners
- Medical consultants
- Healthcare project developers
- NABH-compliant hospital builders

The website should present Wellman Group as a **professional healthcare infrastructure solutions provider**.

# Design Style

Design guidelines:

- Professional healthcare/engineering theme
- Clean white and light blue backgrounds
- Blue / teal medical colors with gradients
- Real OT and hospital photos (images/ot_*.jpeg)
- Glassmorphism effects on hover
- Smooth animations and transitions
- Auto-scrolling marquees for logos and testimonials

Typography:
- Font: Montserrat (Google Fonts) - weights 400, 500, 600, 700, 800, 900
- Headings: Bold with letter-spacing
- Body: Clean readable 1rem base

Card Styles:
- Rounded corners (16-20px radius)
- Subtle shadows with blue tint
- Hover lift effects (translateY + shadow increase)
- Border transitions to primary color

Section Headers (Standardized Design):
- Gradient eyebrow pill badge (linear-gradient cyan to accent blue)
- Bold title (3rem) with highlighted word using gradient text
- Example: "Our **Services**" where Services has gradient text
- Consistent across all sections: About, Services, Materials, MGPS, Team, Testimonials, Why Choose Us

Example layout inspiration:
Modern corporate healthcare engineering websites.
